Hexo是一款基于 Node.js 的静态博客框架,因此可以看出用Hexo搭建个人博客需要使用Node.js、Github申请，使用过github的人应该都知道Git工具。因此我们需要如下几个工具：

  • Node.js
  • Git
  • 申请Github

今天我们都假设这些基本工作都已准备，直接在 Ubuntu 系统上搭建 Hexo 环境，并使用 Material 主题。
